# Pre-Release Checklist

Use this checklist before each planned release.

## Product and Engineering

- [ ] Scope freeze confirmed
- [ ] Critical bugs triaged
- [ ] Feature flags reviewed

## Quality

- [ ] Unit and integration tests passed
- [ ] Smoke tests passed
- [ ] Security-sensitive paths manually verified

## Operations

- [ ] Migration and rollback plan reviewed
- [ ] Monitoring/alerts checked
- [ ] Release communication prepared
