# Incident Response Runbook

Use this for production incident handling.

## Severity Bands

- Sev-1: Major outage/security incident
- Sev-2: Significant degradation
- Sev-3: Minor service issue

## First Response Checklist

1. Confirm impact and scope
2. Assign incident lead
3. Mitigate immediate risk
4. Communicate status
